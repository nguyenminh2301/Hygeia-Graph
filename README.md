# Hygeia Graph

## Description
Project for analyzing and visualizing contracts using graph theory.

## Setup
1. Install dependencies: `pip install -r requirements.txt`
2. Run the app: `streamlit run app.py`

## Structure
- `app.py`: Main Streamlit application.
- `src/hygeia_graph`: Main package code.
- `contracts/`: JSON schemas and contract data.
- `tests/`: Unit tests.
